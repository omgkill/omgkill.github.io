  1. windows系统要求
  • 系统是专业版
  • 系统版本号Windows10 2018 April（Windows 1803）及以上版本
  1. 安装需要的功能
    • 打开控制面板 -> 搜索启动或关闭Windows功能 -> 选择以下功能 -> 点击确定 -> 安装 -> 重启
  2. 更新wsl2
    1. 下载安装
      1. https://wslstorestorage.blob.core.windows.net/wslblob/wsl_update_x64.msi
    2. 在管理员下执行命令
      1. wsl –set-default-version 2
    3. 需要更多的设置参考
      1. https://docs.microsoft.com/zh-cn/windows/wsl/setup/environment
      2. https://docs.microsoft.com/zh-cn/windows/wsl/tutorials/gui-apps
      3. https://docs.microsoft.com/en-us/windows/wsl/install-manual
  3. 更新wsl2内核 - 在管理员下执行命令
    1. 更新：wsl –update
    2. 重启：wsl –shutdown
  4. 安装centos7子系统
    1. 下载地址，对应centos7
      1. https://github.com/wsldl-pg/CentWSL/releases/
    2. 解压到指定目录
      1. 点击centos7.exe,会自动安装
      2. 完成后，我们查看当前子系统
        1. 命令行下执行：wsl -l -v
      3. 设置centos7为默认子系统
        1. wslconfig /s CentOS7
      4. wsl 进入centos7,查看系统信息
        1. cat /etc/redhat-release
  5. 目前为止，windows下linux子系统已安装完成，接下连安装docker

docker

  1. 下载地址
  2. https://www.docker.com/get-started/
  3. 选择wsl2安装
  4. 安装完成后，需要改的配置
  5. 加仓库地址
  6. docker也安装完成

启动docker命令

  • 需要把服id（SERVER_ID）改为自己的服id
