linux

  1. 输出类型

    • /dev/stderr 代表标准错误
    • /dev/stdin 输出可以用 -
    • /dev/stdout 代表标准输出

  2. 输出符号意义 > 表示先清空文件，然后再写内容 >> 表示内容追加到现有文件的尾部

  3. /dev/null 是一个特殊的设备文件，它接收的任何数据都会丢弃

  4. 将脚本内部的文本块进行重定向

    1#! /bin/base
2cat  <<EOF>> log.txt
3LOG FILE HEADER
4this is a test log file
5Function: System statics
6EOF
    • < 从文件读取stdin
  • > 操作符，用于截断模式的文件写入
  • >> 操作追加模式的文件写入

  5. 使用sed替换文本

     11. 替换
 2   sed 's/pattern/replace_string/' file
 3   cat etc/passwd/ | cut -d : -f1,3 | sed 's/:/ - UID: /'
 42. 选项-i 会使得sed用修改后的数据替换原文件

 63. g标记可以使sed执行全局替换
 7   /#g 标记可以使sed替换第N次出现的匹配：
 8   -  $ echo thisthisthisthis | sed 's/this/THIS/2g'

10   - $ echo thisthisthisthis | sed 's/this/THIS/3g'

13   sed 's:text:replace:g'

155. 正则表达式 -- 移除空行 ^$ 匹配空行， /d 执行删除
16    sed '/^$d' file
176. 直接在文件中替换
18    sed 's/PATTERN/replacement' -i filename
19    sed -i 's/\b[0-9]\{3\}\b/NUMBER/g' sed_data.txt
20    其中\b 表示单词边界
21    - 还可以先不替换源文件，新建一个

33 9. 组合多个表达式

  6. 使用awk进行高级文本处理

     1awk 脚本的结构如下：
 2-  awk 'BEGIN{ print "start"} pattern {commands} END {print "end"}' file 
 3-  awk 'BEGIN {statements} {statements} END {end statements}'
 4-  awk "BEGIN {i =0} {i++} END {print i}" filename
 51. 中间{} 默认执行{print}
 62. print 能够接受参数，这些参数以逗号分隔
 7    echo | awk '{var1="v1";var2="v2";var3="v3";print var1,var2,var3;}'
 83. 特殊变量
 9    - NR: 表示记录编号，当awk将行作为记录时，该变量相当于当前行号
10    - NF: 表示字段数量，在处理当前记录时，相当于字段数量，默认的字段分隔符是空格
11    - $0: 该变量包含当前记录的文本内容
12    - $1: 该变量包含第一个字段的文本内容
13    - $2: 该变量包含第二个字段的文本内容
14
153.awk使用正则
16    seq 10 | awk 'BEGIN{print "start"} /6.*/ {print} END{print "end"}'
174. 例子
18    echo -e "line1 f2 f3\nline2 f4 f5\nline3 f6 f7" | \
19    awk '{
20        print "Line no:"NR",No of fields:"NF, "$0="$0,
21        "$1="$1,"$2="$2,"$3="$3
22    }'
235. 打印每一行的第二和第三个字段
24    awK '{print $3,$2}' file
256. 我们可以使用NR统计文件行数
26    awk 'END {print NR}' file
277. 借助选项-v， 我们可以将外部值传递给awk：
28    $VAR=10000
29    $ echo | awk -v VARIABLE = $VAR '{print VARIABLE}'
308. 还有一种灵活的方法可以将多个外部变量传递给awk
31    $ var1 = "Variable1"; var2 = "Variable2"
32    $ echo | awk '{print v1, v2}' v1=$var1 v2 = $var2
33    当输入来自于文件而非标准输入时，使用下列命令
34        $ awk '{print v1,v2}' v1=$var1 v2=$var2 filename
359. 用getline读取行
36    seq 5 | awk 'BEGIN {getline;print "Read ahead first line", $0}' {print $0}
3710. 使用过滤模式对awk处理的行进行过滤
38    $ awk 'NR < 5' # 行号小于5的行
39    $ awk 'NR==1, NR==4' # 行号在1到5之间的行
40    $ awk '/linux/' #包含模式为linux的行（可以用正则表达式来指定模式）
41    $ awk '!/linux/' # 不包含模式为linux的行
4211. 设置字段分隔符
43    awk -F: '{print $NF}' /etc/passwd

5014. 使用if
51	netstat -tanp  | awk '{if ($2 > 0 || $3 > 0) print $0}'
5215. awk内建的字符串处理函数
53    - length(string) : 返回字符串string的长度
54    - index(string,serach_string) : 返回search_string 在字符串string出现的位置
55    - split(string, array, delimiter):以delimiter作为分隔符，分割字符串string，将生成的字符串存入数组array
56    - substr(string, start-position, end-position) : 返回字符串string中以start-position 和 end-position 作为起止位置的字符串
57    - sub(regex, replancement_str, string): 将正则表达式regex匹配到的第一处内容替换成replacement_str
58    - gsub(regex, replacement_str, string) : 和sub()类似。不过该函数会替换正则表达式 regex匹配到的所有内容
59    - match(regex, string) : 检查正则表达式regex是否能够在字符串string中找到匹配。如果能够找到，返回非0值；否则，返回0. match() 有两个相关的特殊变量，分别是RSTART 和 RLENGTH。变量RSTATRTv包含了匹配内容的起止位置，而变量RLENGTH包含了匹配内容的长度

  7. 用tr进行转换

     1tr 是translate（转换）的简写
 2    1. tr [options] set1 set2
 3    2. 要将输入中的字符由大写转换为小写，可以使用下面的命令
 4        echo "HELLO WHO IS THIS" | tr 'A-Z' 'a-z'
 5        hello who is this
 6    3. tr 进行加密和解密
 7        echo 12345 | tr '0-9' '9876543210'
 8            87654 # 已加密
 9        echo 87654 | tr '9876543210' '0-9'
10            12345 # 已解密
11    4. 用tr删除字符
12        cat file.txt | tr -d '[set1]'
13        例如： 
14           echo "Hello 123 world 456" | tr -d '0-9'
15           hello world
16    5. 字符组补集
17        tr -c \[set1\] [set2]
18        - 从输入文本中删除不在补集中的所有字符
19          echo hello 1 char 2 next 4 | tr -d -c '0-9 \n'
20          124
21        - 不在set1中的字符替换成空格
22          echo hello 1 char 2 next 4 | tr -c '0-9' ''
23              1    2   4
24        - 删除多余的空格
25          tr -s ' '
26        - 删除多余的换行符
27          tr -s '\n'
28        - cat sum.txt | echo $[ $(tr '\n' '+' ) 0 ]
29
30        - 包含字母和数字的文件， 计算数字之和
31          cat test.txt | tr -d [a-z] | echo "total: $[$(tr ' ' '+'')]"
32        - 字符类
33            alnum : 字母和数字
34            alpha : 字母
35            cntrl : 控制（非打印）字符
36            digit : 数字
37            graph : 图形字符
38            lower : 小写字母
39            print : 可打印字符
40            punct : 标点符号
41            space : 空白字符
42            upper : 大写字母
43            xdigit : 十六进制字符
44
45            可以按照下面的方式选择所需的字符类
46            tr [:class:] [:class:]
47            例如 : 
48                tr '[:lower:]' '[:upper:]'
